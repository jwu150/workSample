Column Filters

First select a data type below, then click on the filter icon to see the pop-up filter

A column filter icon is attached to the table header when appropriate. On activation of the filter it will show the corresponding filter with correct operators and controls. Users filled out the popups to construct the query.

  • Query combined filters with support for AND or OR operation
  • Forms have built in validation (e.g., To date must be later than From date)
  • Keyboard fully operational with focus cycle inside each popup
  • On Submit,the expression is built with all filters and query submit to server via Spel Expression
